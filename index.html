Accessibility

Automated tool for the accessibility's Analysis

The Tool is able to effect both the validation tests of HTML that of accessibility with its free web integration tools as a-checker and the tool of w3c that otherwise must be directly interfaced via web. It is a console application that is able to interact automatically with these web services and provide a log of the results is txt or html format (most advanced), distinguishing between errors and warnings. The functions can also be used as Unit Test in a cycle of Continuous Integration
